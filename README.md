# Data4Good Volunteering: Git Advanced

## Exercise Change & Publish
For this exercise, the **team members take turns**. The following workflow should be followed by each team member. Each team member should have *practiced the flow at least 2-3 times*.

---

## Workflow

1. download remote state `git fetch`

2. pull changes into your local branch: `git pull`

3. find the next task (see below) marked with `//`

4. make the change locally

5. commit and push changes
